# 暗色模式扩展

为任意网站提供舒适的暗色模式体验，使用智能过滤算法，支持白名单和黑名单管理。

## 功能特点

### 🌙 智能暗色模式
- 使用 CSS 滤镜算法实现智能暗色模式
- 自动处理图片和视频，保持正确的色彩
- 平滑的视觉效果，不影响用户体验

### 🎨 可自定义调节
- **亮度**：50% - 150%
- **对比度**：50% - 150%
- **反转强度**：0% - 100%
- 实时预览调节效果

### 🏷️ 黑白名单管理
- **白名单**：网站始终启用暗色模式（最高优先级）
- **黑名单**：网站从不启用暗色模式（第二优先级）
- 当前网站状态实时显示
- 一键添加当前网站到黑/白名单

### 🎯 系统主题跟随
- 可选跟随系统的亮色/暗色主题自动切换
- 完美适配系统外观设置

### 🎨 莫奈风格 UI
- 优雅的暗色调配色
- 柔和的渐变效果
- 流畅的交互动画
- 响应式设计

## 安装方法

### Chrome / Edge 浏览器

1. 下载或克隆此项目
2. 打开 Chrome/Edge 浏览器
3. 访问 `chrome://extensions/`（Chrome）或 `edge://extensions/`（Edge）
4. 开启右上角的"开发者模式"
5. 点击"加载已解压的扩展程序"
6. 选择项目根目录
7. 完成！

### Firefox 浏览器

Firefox 版本即将推出，敬请期待。

## 使用说明

### 快速使用

1. 点击浏览器工具栏中的插件图标
2. 切换"启用暗色模式"开关
3. 享受舒适的暗色模式！

### 详细设置

点击插件图标打开小窗，您可以：

1. **启用/禁用暗色模式**
2. **调节亮度、对比度、反转强度**
3. **查看当前网站的黑白名单状态**
4. **一键添加当前网站到白名单或黑名单**
5. **点击设置按钮打开完整设置页面**

### 完整设置页面

在完整设置页面中，您可以：

1. **通用设置**：配置默认启用、系统主题跟随
2. **外观设置**：设置默认的滤镜参数
3. **网站管理**：管理白名单和黑名单

## 黑白名单优先级

黑白名单的判断优先级如下（从高到低）：

1. **黑名单** → 始终禁用暗色模式
2. **白名单** → 始终启用暗色模式
3. **总开关** → 是否默认启用
4. **系统主题** → 是否跟随系统

即使关闭了总开关，白名单中的网站仍会启用暗色模式！

## 卸载

## 技术实现

### 核心技术
- **Manifest V3**：最新的 Chrome 扩展标准
- **Content Script**：注入网页的脚本
- **Service Worker**：后台服务
- **Storage Sync**：跨设备设置同步

### 智能过滤算法
```javascript
// 使用 CSS filter 实现
filter: invert(1) hue-rotate(180deg) brightness(0.9);
```

### 图片处理
- 自动识别图片、视频、SVG、Canvas 等媒体元素
- 对这些元素再次反转，恢复正确的色彩
- 支持背景图片识别
